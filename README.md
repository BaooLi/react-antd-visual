# 基于react+antd的后台管理系统可视化生成工具

## 涉及技术

### 脚手架

- create-react-app

### 技术

- react
- antd
- react-dnd
- react-hot-reloader

## 运行步骤

```
git clone https://github.com/lixuejiang/react-antd-visual.git
cd react-antd-visual
yarn
yarn start
```
